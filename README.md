# Robintrack API

Unofficial node client for accessing the APIs called on Robintrack.net.

## Installation

```sh
npm i -S @joemccann/robintrack-api
```

## Usage

```js
const rta = require('robintrack-api')
```

## Tests

```sh
npm i -D
npm test
```
